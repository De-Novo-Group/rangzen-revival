The Feed

Your feed shows messages from the mesh network:

  • Green bar = From a friend (trusted)
  • Yellow bar = Neutral source
  • Red bar = Unknown/low trust

Pull down to refresh and scan for nearby peers.

Tap the heart to bookmark a message locally.

Tips

  • Be patient: Messages take time to spread through the network.
  • Stay near people: The more users nearby, the faster messages spread.
  • Keep Bluetooth on: This is how devices discover each other.
  • Check Settings: Make sure "Enable Murmur" is ON.
